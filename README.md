# Business Vault

**Under every business is a layer of digital infrastructure. Domains. Hosting. Logins. Contracts. Can you access them quickly and easily?**

Business Vault is a unified documentation platform for business assets, helping companies protect their digital infrastructure and intellectual property.

## Features

- **Physical Assets**: Track equipment, vehicles, and property
- **Digital Infrastructure**: Document domains, hosting, and credentials
- **Contracts**: Manage agreements and renewal dates
- **Subscriptions**: Monitor SaaS and service subscriptions
- **Key Contacts**: Maintain critical business relationships
- **PDF Export**: Generate comprehensive asset reports

## Quick Start

1. Clone the repository
2. Copy `.env.example` to `.env.local` and add your Supabase credentials
3. Run `npm install`
4. Run `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase
